- Мобильное SPA веб-приложение
- Полная поддержка работы на мобльных устройствах
- Фронт Vue + TS + Pinia + Vue-Router
- Сервер Node.js + Fastify + Sequelize + MySQL
- Авторизация по JWT-токену (будет перенесена на access + refresh токены)
- Плавные анимации используя возможности Vue-transition и CSS
- Код по запросу на собеседовании
