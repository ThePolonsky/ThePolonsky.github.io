## 🔑 Особенности

- Адаптивный лендинг (desktop, tablet, mobile)
- Поддержка светлой/тёмной темы (через `data-theme` + `localStorage`)
- Анимации на **GSAP**
- Слайдер отзывов на **Swiper.js**
- Аккордеон FAQ
- Раскрывающийся grid в блоке Solutions
